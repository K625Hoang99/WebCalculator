Step 1: Basic Button Element

Start with a basic HTML button element:

                
<button>Button</button>

Step 2: Add CSS Width and Height

Set the width and height of the button:

button {
    width: 100px;
    height: 100px;
}

Step 3: Add Border Radius

Make the button rounded by setting the border radius:

button {
    border-radius: 50px;
}

Step 4: Remove Border

Remove the default border:

button {
    border: none;
}

Step 5: Set Background Color

Change the background color to a light grey:

button {
    background-color: hsl(0, 0%, 77%);
}

Final Result

Your final button styling should look like this:

button {
    width: 100px;
    height: 100px;
    border-radius: 50px;
    border: none;
    background-color: hsl(0, 0%, 77%);
}
